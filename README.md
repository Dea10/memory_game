# Memory Game App

Memory Game with Pokemón theme.

Give a try: https://dea10.github.io/memory_game/
### Technologies
* Frontend
    * React
    * JavaScript
    * TypeScript
    * Bootstrap
    * Sass
    * Flexbox
* Backend
    * Node
    * JavaScript
    * Axios
* Database
    * MongoDB
* Deployment
    * Github pages
    * Heroku
    * MongoDB Cloud
